Kebijakan Privasi

Terakhir diperbarui: 11 Desember 2025

Selamat datang di aplikasi The Icon Connect. Aplikasi ini dikembangkan untuk membantu pengelolaan lingkungan dan memudahkan komunikasi antar warga di perumahan The Icon Acropolis.

1. Informasi yang Kami Kumpulkan

Agar aplikasi dapat berfungsi dengan baik sebagai sarana manajemen perumahan, kami mengumpulkan beberapa informasi berikut:

  • Informasi Identitas: Nama lengkap dan alamat unit rumah (Blok/Nomor).
  • Informasi Kontak: Nomor telepon/WhatsApp untuk verifikasi warga.
  • Informasi Perangkat: ID Perangkat untuk keperluan notifikasi (push notifications).
  • Data Aktivitas: Riwayat pembayaran iuran atau laporan keluhan (jika fitur tersedia).

2. Bagaimana Kami Menggunakan Informasi Anda

Data yang dikumpulkan digunakan semata-mata untuk:

  • Memverifikasi bahwa pengguna adalah benar warga The Icon Acropolis.
  • Mengirimkan informasi penting atau pengumuman perumahan.
  • Mengelola data keamanan dan ketertiban lingkungan.
  • Meningkatkan layanan aplikasi.

3. Keamanan Data

Kami sangat menghargai privasi Anda. Kami tidak akan menjual, menukar, atau menyewakan informasi identitas pribadi pengguna kepada pihak ketiga. Akses data dibatasi hanya untuk pengurus lingkungan yang berwenang.

4. Izin Akses (Permissions)

Aplikasi ini mungkin meminta izin akses tertentu di perangkat Anda, seperti:

  • Internet: Untuk terhubung ke server database.
  • Penyimpanan/Kamera: Jika Anda menggunakan fitur pelaporan yang membutuhkan upload foto bukti pembayaran atau kejadian.
